# Feature: Напоминания для привычек

## Problem
Пользователи забывают выполнять привычки в нужное время. Нет системы напоминаний.

## Description
Добавить возможность устанавливать напоминания для привычек — push-уведомления в определённое время.

## Notes
- Должно работать локально (без сервера)
- Простой UI — минимум настроек
- Интеграция с iOS Notifications
