# QA Checklist: Напоминания для привычек

**Feature:** Habit Reminders
**Date:** 2025-12-26
**Tester:** QA Agent

---

## Acceptance Criteria

| # | Критерий | Статус |
|---|----------|--------|
| AC-1 | Пользователь может включить/выключить напоминание | ⬜ |
| AC-2 | Пользователь может выбрать время напоминания | ⬜ |
| AC-3 | Push-уведомление приходит в указанное время | ⬜ |

## Test Cases

### TC-001: Открытие Habit Detail Sheet

**Preconditions:**
- Пользователь авторизован
- Есть хотя бы одна привычка

**Steps:**
1. Открыть раздел Habits
2. Тапнуть на любую привычку

**Expected:**
- Открывается sheet с деталями привычки
- Видна иконка, название, период, streak
- Есть toggle "Reminder"
- Есть кнопка "Delete Habit"

**Result:** ⬜ PASS / ⬜ FAIL

---

### TC-002: Включение напоминания (первый раз)

**Preconditions:**
- Notification permission не запрошен ранее

**Steps:**
1. Открыть detail sheet привычки
2. Включить toggle "Reminder"

**Expected:**
- Появляется системный запрос permission
- При согласии — появляется time picker
- При отказе — toggle выключается, показан alert

**Result:** ⬜ PASS / ⬜ FAIL

---

### TC-003: Выбор времени напоминания

**Preconditions:**
- Notification permission granted

**Steps:**
1. Включить reminder toggle
2. Изменить время на 14:30

**Expected:**
- Time picker показывает выбранное время
- Время сохраняется при закрытии sheet
- В списке привычек показан badge 🔔 14:30

**Result:** ⬜ PASS / ⬜ FAIL

---

### TC-004: Получение push-уведомления

**Preconditions:**
- Reminder включён на время через 1-2 минуты

**Steps:**
1. Установить напоминание на ближайшее время
2. Свернуть приложение
3. Дождаться времени

**Expected:**
- Приходит push-уведомление
- Title: "Atoma"
- Body: "[иконка] [название привычки]"
- Subtitle: "Time to build your routine!"

**Result:** ⬜ PASS / ⬜ FAIL

---

### TC-005: Тап по уведомлению

**Steps:**
1. Получить уведомление
2. Тапнуть на него

**Expected:**
- Открывается приложение

**Result:** ⬜ PASS / ⬜ FAIL

---

### TC-006: Уведомление при открытом приложении

**Steps:**
1. Установить напоминание на ближайшее время
2. Оставить приложение открытым
3. Дождаться времени

**Expected:**
- Уведомление показывается как banner внутри приложения

**Result:** ⬜ PASS / ⬜ FAIL

---

### TC-007: Отключение напоминания

**Steps:**
1. Открыть привычку с включённым напоминанием
2. Выключить toggle "Reminder"

**Expected:**
- Time picker скрывается
- Badge в списке пропадает
- Уведомление отменяется (не приходит в указанное время)

**Result:** ⬜ PASS / ⬜ FAIL

---

### TC-008: Удаление привычки с напоминанием

**Steps:**
1. Создать привычку с напоминанием
2. Удалить привычку

**Expected:**
- Привычка удалена
- Запланированное уведомление отменено

**Result:** ⬜ PASS / ⬜ FAIL

---

### TC-009: Сохранение после перезапуска

**Steps:**
1. Создать привычку с напоминанием на 10:00
2. Закрыть приложение (kill)
3. Открыть приложение

**Expected:**
- Привычка сохранена
- Reminder toggle включён
- Время 10:00 сохранено
- Badge показывается в списке

**Result:** ⬜ PASS / ⬜ FAIL

---

### TC-010: Offline режим

**Steps:**
1. Включить airplane mode
2. Создать привычку
3. Установить напоминание

**Expected:**
- Напоминание устанавливается (локально)
- Уведомление придёт в указанное время

**Result:** ⬜ PASS / ⬜ FAIL

---

### TC-011: Permission denied

**Steps:**
1. Отклонить permission при первом запросе
2. Попробовать включить reminder

**Expected:**
- Показан alert с кнопкой "Open Settings"
- Кнопка ведёт в Settings приложения
- Toggle остаётся выключенным

**Result:** ⬜ PASS / ⬜ FAIL

## Edge Cases

| Case | Expected | Status |
|------|----------|--------|
| Очень длинное название привычки | Текст обрезается в уведомлении | ⬜ |
| Emoji в названии привычки | Emoji отображается корректно | ⬜ |
| Midnight time (00:00) | Напоминание приходит в полночь | ⬜ |
| Несколько привычек с одинаковым временем | Все уведомления приходят | ⬜ |
| Изменение timezone | Время пересчитывается | ⬜ |
